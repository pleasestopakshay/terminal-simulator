COMMON QUESTIONS

▶ What commands can I type?
You can type any text and the terminal will simulate realistic command responses. Try commands like 'ls', 'pwd', 'whoami', or just type random text to see system responses.
▶ How do I execute commands?
Simply press Enter after typing your command, or wait a moment and the system will auto-execute. You can also use the quick command buttons on the left.
▶ Can I delete what I typed?
Yes! Use the Backspace key to delete characters you've typed. The terminal behaves like a real command line interface.
▶ What's the difference between SLOW and FAST mode?
SLOW mode adds realistic delays to make the terminal feel more authentic, while FAST mode executes commands immediately for quick demonstrations.
